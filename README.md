# Bookloggd-App

## Description

Web application built with Spring Boot for managing books to read, featuring a REST API, user authentication, and more. For study purpose only.

## Features

- Manually register books (title, author, ISBN, genre, summary, reading status)
- Search books through the Google Books API using ISBN/title for auto-filling
- Manage reading status: “Want to Read”, “Reading”, “Read”
- Filter and search books by various criteria
- User authentication
- Visually appealing interface displaying book covers
- Example endpoint: GET /api/tasks

## Technologies Used

- Java 17
- Spring Boot
- Maven
- H2 (dev), PostgreSQL (prod)
- React, Axios
